# 후제공방 애드온 (HoojeStudio)

> **아즈샤라 서버 주문 제작 서비스 전용 도우미 애드온**  
> hooje.pro | 제작자: 후제

---

## 애드온 소개

후제공방 애드온은 World of Warcraft 아즈샤라 서버의 주문 제작 서비스 이용자를 위해 만들어진 UI 도우미입니다.  
"재료 준비 → 제작자 확인 → 주문 의뢰" 전체 흐름을 하나의 창 안에서 처리할 수 있도록 설계했습니다.  
직접 찾아다니고, 직접 검색하고, 직접 입력해야 했던 모든 단계를 최대한 자동화했습니다.

---

## 핵심 기능

### 1. 자동 감지 (Auto Detection)

주문 제작 NPC 창에서 아이템을 클릭하는 순간, 애드온이 자동으로 해당 아이템을 인식합니다.

- 아이템 이름이 오토 패널에 자동으로 표시됩니다
- 해당 아이템을 제작할 수 있는 **담당 제작자**가 즉시 표시됩니다
- 제작자마다 고유한 클래스 컬러로 구분되어 한눈에 알아볼 수 있습니다
- 후제공방 미지원 아이템은 "후제공방 미지원"으로 표시되어 혼동을 방지합니다

### 2. 받는이 자동 입력

주문 제작 의뢰창의 "받는이" 필드에 제작자 이름을 직접 타이핑할 필요가 없습니다.

- **[받는이에 입력]** 버튼 한 번 클릭으로 제작자 이름이 자동으로 입력됩니다
- 입력 완료 시 화면 하단에 토스트 알림이 표시됩니다
- 주문 제작창이 열려있지 않은 경우 안내 메시지로 알려줍니다

### 3. 재료 목록 자동 표시

아이템이 감지되면 필요한 재료 목록이 자동으로 펼쳐집니다.

- 재료명과 필요 수량이 함께 표시됩니다
- **금성(★) 표시 재료**는 별도로 강조 표시되어, 반드시 금성 등급으로 구매해야 함을 알려줍니다
- 금성 재료를 일반 등급으로 구매하는 실수를 방지하기 위한 기능입니다

### 4. 경매장 자동 검색 연동

재료 목록에서 재료 이름을 클릭하면 경매장 검색창이 자동으로 채워집니다.

- 경매장을 따로 열어 이름을 타이핑할 필요 없이, 클릭 한 번으로 검색됩니다
- 경매장이 열려있지 않은 경우 안내 메시지를 표시합니다
- 경매장 UI 구조가 달라도 대응 가능하도록 다중 경로 탐색을 내장했습니다
- 자동 입력이 불가능한 경우 복사 팝업을 띄워 붙여넣기로 대체합니다

### 5. 경매장 모드 (AH 모드)

경매장을 사용하는 동안 애드온 창이 방해가 되지 않도록 설계했습니다.

- **[경매장 숨김]** 버튼을 누르면 메인 창이 사라지고 경매장이 화면을 넓게 사용할 수 있습니다
- 경매장 화면 우측에 작은 **사이드 버튼**이 남아있어, 언제든지 다시 열 수 있습니다
- 경매장을 닫으면 자동으로 메인 창이 복원됩니다
- 숨김 상태에서도 재료 클릭 → 경매장 검색은 정상 동작합니다

### 6. 매뉴얼 패널 (아이템 목록 탐색)

자동 감지 없이도 직접 아이템을 찾아볼 수 있는 브라우저 패널입니다.

- 천 / 가죽 / 사슬 / 판금 / 무기류 / 공통(망토·반지·목걸이) / 전문기술 장비 / 기계공학 / PVP 무기 / PVP 공통 / PVP 천·가죽·사슬·판금 — 전 카테고리를 모두 수록했습니다
- 카테고리를 클릭하면 해당 카테고리만 펼쳐지는 **아코디언 방식**으로 구성되어 있습니다
- **실시간 검색** 기능으로 아이템 이름 일부만 입력해도 즉시 필터링됩니다
- 검색 시 스크롤이 자동으로 최상단으로 이동합니다
- 매뉴얼에서 아이템을 클릭하면 오토 패널이 해당 아이템으로 동기화됩니다
- 오토 패널의 아이템을 클릭하면 반대로 매뉴얼에서 해당 위치로 이동합니다

### 7. How-to 가이드 (사용 방법 안내)

처음 사용하는 분도 헤매지 않도록 창 안에 사용 방법을 직접 내장했습니다.

**재료 준비 탭:**

- Step 1 — 후제공방 창에서 원하는 아이템 선택 또는 검색
- Step 2 — 필요한 재료 이름 클릭 → 경매장 자동 검색 / 금성 재료는 반드시 금성으로 구매
- Step 3 — 모든 재료 준비 완료 후 제작 의뢰 단계로 이동

**제작 의뢰 탭:**

- Step 0 (선택) — 하단 버튼으로 주소 복사 → 브라우저에서 제작자 실시간 접속 여부 확인
- Step 1 — 주문 제작 의뢰창에서 아이템 검색 후 클릭 → 후제공방 창에 자동 감지
- Step 2 — [받는이에 입력] 버튼 클릭 → 제작자 이름 자동 입력
- Step 3 — 재료 전부 금성인지 확인 후 주문 의뢰
- Step 4 (선택) — [알림 전송] 버튼 → 링크 복사 → 브라우저 붙여넣기 → 제작자에게 실시간 알림 전송

### 8. 알림 전송

주문을 넣은 뒤 제작자에게 직접 알림을 보낼 수 있습니다.

- **[알림 전송]** 버튼을 누르면 알림 링크가 복사됩니다
- 브라우저에 붙여넣기하면 hooje.pro를 통해 해당 제작자에게 실시간 알림이 전달됩니다
- 게임 내 귓속말 없이도 주문 접수 사실을 제작자에게 알릴 수 있습니다

### 9. 실시간 접속 여부 확인

제작 의뢰 전에 제작자가 현재 접속 중인지 확인할 수 있습니다.

- 후제공방 창 하단 버튼으로 제작자 페이지 주소가 복사됩니다
- 브라우저에 붙여넣기하면 hooje.pro에서 해당 제작자의 실시간 접속 상태를 확인할 수 있습니다
- 접속 중이 아닌 제작자에게 주문을 넣는 혼선을 방지할 수 있습니다

## UI / 편의성 설계

### 창 위치 저장 및 잠금

- 창을 원하는 위치로 드래그한 뒤 위치가 자동으로 저장됩니다
- 재접속 후에도 마지막 위치 그대로 열립니다
- **잠금 버튼**으로 창을 고정하면 실수로 움직이는 것을 방지할 수 있습니다

### 미니맵 버튼

- 미니맵 주변에 후제공방 버튼이 생성됩니다
- 클릭 한 번으로 창을 열고 닫을 수 있습니다
- NPC나 경매장을 열지 않아도 언제든지 접근 가능합니다

### 슬래시 커맨드 (/hj)

채팅창에 `/hj`를 입력하면 언제 어디서나 창을 열고 닫을 수 있습니다.

- 주문 제작 NPC 창이나 경매장이 열려있지 않아도 작동합니다
- 추가 커맨드:
  - `/hj reset` — 창 위치 초기화
  - `/hj center` — 화면 중앙으로 이동
  - `/hj debug` — 디버그 정보 출력

### 툴팁 정상 표시

- 모든 재료 이름에 마우스를 올리면 와우헤드 아이템 툴팁이 정상 표시됩니다
- 애드온 창이 툴팁을 가리지 않도록 프레임 레이어를 의도적으로 설계했습니다

### 내용 초기화

- **[내용 초기화]** 버튼으로 검색창과 선택 아이템을 한 번에 초기화할 수 있습니다
- 오토 패널도 기본 안내 문구로 돌아갑니다

### 텍스트 클리핑

- How-to 안내 텍스트가 지정된 영역 밖으로 넘치지 않도록 설계되어 있습니다
- 긴 내용은 스크롤로 확인할 수 있습니다

## 시스템 요구사항

- **게임 버전**: World of Warcraft Retail (Interface 120005 이상)
- **서버**: 아즈샤라 서버 전용 (제작자 DB가 아즈샤라 서버 기준으로 구성됩니다)

---

## 설치 방법

1. [CurseForge](https://www.curseforge.com/wow/addons/hoojestudio)에서 최신 버전 다운로드
2. 압축 해제 후 `HoojeStudio` 폴더를 `World of Warcraft/_retail_/Interface/AddOns/`에 붙여넣기
3. 게임 실행 후 애드온 목록에서 **HoojeStudio** 활성화
